# 專案說明

## 專案目的
研究級智慧嬰語翻譯機 2.0：以公開資料集（donateacry-corpus）訓練真實聲學分類模型（log-mel + CNN → ONNX），判讀 0–6 個月嬰兒哭聲的可能需求，Streamlit 介面，目標發布至 GitHub + Streamlit Community Cloud。前代為 033（啟發式規則 PWA），本版的核心價值是「真模型 + 誠實評估報告」。

## 技術棧 / 啟動方式
- Python + Streamlit（介面）、PyTorch（僅訓練）、onnxruntime + librosa（推論）
- 固定使用 `.venv`；安裝：`requirements.txt`（App）、`requirements-train.txt`（訓練另加）
- 啟動：`.venv\Scripts\python -m streamlit run app.py`
- 訓練重現：`.venv\Scripts\python src\train.py`（資料集需先在 `data\donateacry-corpus\`）

## 重要邊界
- 照護輔助工具，非醫療診斷；警訊（發燒、呼吸困難、發青、難喚醒等）一律導向就醫。
- 適用月齡 0–6 個月；使用者自己的孩子（2025-07-21 生）已過適用期，本專案定位為研究級作品與對外分享。
